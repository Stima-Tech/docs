# Base URL, API Key 連接說明

Apertis 提供 `BASE_URL` 與 `API Key` 連接至自建專案，不同的應用程式適用於不同的 `BASE_URL`。

### BASE_URL

#### OpenAI Compatible Endpoints
- **[https://api.apertis.ai/v1](https://api.apertis.ai/v1)** - Base URL for OpenAI SDK
- **[https://api.apertis.ai/v1/chat/completions](https://api.apertis.ai/v1/chat/completions)** - Chat Completions API
- **[https://api.apertis.ai/v1/responses](https://api.apertis.ai/v1/responses)** - OpenAI Responses API

#### Anthropic Compatible Endpoint
- **[https://api.apertis.ai/messages](https://api.apertis.ai/messages)** - Anthropic Messages API

#### Other Endpoints
- **[https://api.apertis.ai/api/models](https://api.apertis.ai/api/models)** - List available models (no auth required)

### API Key
請至 **[Apertis Dashboard](https://apertis.ai/token)** 取得 `API Key`。
